Component In React : 
------------------------
    ->single single UI blocks is called Component 
    Types of Component : 
    ----------------------
        1. Class Based Component (CBC)
                    ->we can Declare CBC using Class Keyword 
                    ->CBC Render() is present 
                    ->CBC constructor , super keyword , this keyword  is present 
                    ->In CBC life cycles methods are present 
                    ->In CBC states are present so CBC also called as a state full Component
                
        2. Function Based Component(FBC) 
                    ->we can Declare FBC  using Function Keyword or arrow Function
                    ->FBC Render() is not present 
                    ->In FBC constructor , super keyword , this keyword  is not present 
                    ->In FBC life cycles methods are not present 
                    ->In FBC states are not present so FBC also called as a state less Component




    state In React  : 
    ------------------
        ->states are used to hold data 
        ->states are mutable 
        ->states present only in class based component
